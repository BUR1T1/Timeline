# Timeline API

API REST desenvolvida em Java com Spring Boot para gerenciamento de registros organizados em timelines por usuário.

## Descrição

O projeto Timeline tem como objetivo permitir que usuários criem e gerenciem registros dentro de uma timeline pessoal, permitindo organizar informações de forma estruturada, com controle de acesso e operações completas de CRUD.

A aplicação segue uma arquitetura baseada em camadas (Controller, Service, Repository), garantindo separação de responsabilidades e facilidade de manutenção.

## Tecnologias utilizadas

- Java 17  
- Spring Boot  
- Spring Data JPA  
- Spring Security  
- PostgreSQL  
- Maven  
- Docker  

## Funcionalidades

- Cadastro e autenticação de usuários  
- Criação de timelines  
- Cadastro de registros dentro de uma timeline  
- Atualização de registros  
- Exclusão lógica (soft delete)  
- Validação de acesso por usuário  
- Integração com banco de dados PostgreSQL  
- Execução via Docker
